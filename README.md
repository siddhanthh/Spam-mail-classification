# Spam Mail Classifier

An end-to-end Machine Learning web application designed to identify spam emails with high accuracy. This project utilizes a custom PyTorch Feed-Forward Neural Network trained on the Spambase dataset, integrated with a clean and interactive Streamlit web interface.

## Features
- **Real-time Inference**: Paste any email or raw text directly into the web interface and get instantaneous predictions.
- **Robust Feature Extraction**: Automates the extraction of 57 distinct text features (word frequencies, character frequencies, and capital run-lengths) matching the Spambase specification.
- **Visual Confidence Metrics**: Evaluates predictions with an intuitive confidence progress bar.
- **Standalone Design**: Includes separate, decoupled scripts for both model training and application serving.

## Project Architecture
- `app.py`: The main Streamlit web application. Handles the UI, loads the trained model, parses user input, and displays predictions.
- `train.py`: Contains the model architecture (`nn.Module`), data loading, PyTorch training loop, and evaluation logic.
- `feature_extractor.py`: Utility module mapping raw strings to the 57-dimensional feature vectors required by the model.
- `model.pth`: The serialized PyTorch model weights (generated by running `train.py`).
- `data/spambase_csv.csv`: The dataset used to train the neural network.

## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/siddhanthh/Spam-mail-classification.git
   cd Spam-mail-classification
   ```

2. **Create a virtual environment (Recommended):**
   ```bash
   python -m venv env
   
   # Windows
   .\env\Scripts\activate
   
   # macOS/Linux
   source env/bin/activate
   ```

3. **Install the dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

## Usage

**Running the Web Application:**
To launch the interactive Streamlit interface, run:
```bash
streamlit run app.py
```
This will open a local web server (usually at `http://localhost:8501`) where you can paste emails and check for spam.

**Retraining the Model:**
If you want to train the model from scratch or experiment with the neural network architecture, run:
```bash
python train.py
```
This will execute the training loop for 500 epochs, print the evaluation metrics, and overwrite `model.pth`.

## Model Performance
The current Neural Network achieves the following metrics on the 20% hold-out test set:
- **Accuracy**: 94.68%
- **Precision**: 93.83%
- **Recall**: 93.59%
- **F1-Score**: 93.71%
